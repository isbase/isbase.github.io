最近用一个 vue 框架出现的问题

Author: Today Date: May 20, 2020  0:00:00 Category: html,css,h5

一 : Failed to resolve async component default: Error: Cannot find module

今天接到一个项目报这个错.

  • vue-router.esm.js:16 [vue-router] Failed to resolve async component default: Error: Cannot find module ‘@/views/report/rank/index’

解决办法:

  1. 安装syntax-dynamic-import依赖。
  2. 把store/modules/premission.js中的
    return () => import(@/views/${view})改成
    return resolve => require([@/views/${view}], resolve)就可以了

先选择第一种吧.

二: TypeError: mkdirp is not a function

我本地是用 n 来管理 node 版本的.
最后解决办法是:
sudo n 切换了 node 版本. 之后错误就消失了.
