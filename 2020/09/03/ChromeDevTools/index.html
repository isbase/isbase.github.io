Chrome DevTools 记录

Command

  • 打开 Command 命令,可以看看里面都有什么操作
  • Cmd + Shift + P

DOM 截图

  • 特定位置截图
  • Screenshot Capture node screenshot
  • 全屏截图
  • Screenshot Capture full size screenshot

编辑网页内任意内容

  • 命令
  • document.body.contentEditable="true"
  • document.designMode = 'on'

Copy && Save

  • 可以通过全局的方法 copy() 在 console 里 copy 任何你能拿到的资源
  • 只需要右击它，并选择 “Store as global variable”选项
  • 第一次使用的话，它会创建一个名为 temp1 的变量，第二次创建 temp2，第三次 …

DevTools Theme

  • 开启 Allow custom UI theme
  • chrome://flags/#enable-devtools-experiments # 启用实验功能
  • 控制台中使用快捷键F1打开设置，切换到Experiments 选项
  • 启用Allow custom UI themes
