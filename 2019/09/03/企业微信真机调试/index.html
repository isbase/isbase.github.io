企业微信真机调试

一. 微信企业应用真机调试.

  • 微信开发这工具,必须是0.5 - 0.7,这里提供一个 0.7 版本的下载地址.
mac：http://dldir1.qq.com/WechatWebDev/release/0.7.0/wechat_web_devtools_0.7.0.dmg
  • 需要开启一个选项 . 企业微信后台 -> 我的企业->微信插件->开发者工具 选项里打钩

开发者工具 开启后，当前管理员可在工具内模拟微信网页，方便开发与调试。web开发工具

  • 打开手机的 USB 调试功能
  • 如果报错什么 vw.js 啥的直接进入 ~/Library/Application Support , 删除微信 web 开发者工具目录.

需要agentConfig的接口列表

所以没有使用以下接口就不需要调用 agentConfig 接口了

  • selectExternalContact
  • openUserProfile
  • thirdPartyOpenPage
  • getCurExternalContact
